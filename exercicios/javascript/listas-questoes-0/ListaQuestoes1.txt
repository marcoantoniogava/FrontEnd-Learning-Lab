1. Timer com Promise

Cria uma função contagemRegressiva(segundos) que imprime no console a contagem (3... 2... 1... 🚀) usando await esperar(ms) a cada segundo.

Dica: usa a função esperar que tu já fez.

2. Encadeando Promises

Cria uma Promise que começa com o número 2, e:

multiplica por 3,

depois soma 10,

depois mostra o resultado no console.

Faz primeiro com .then() encadeando, depois com async/await.

3. Fetch básico

Usa o fetch pra pegar 10 posts da API https://jsonplaceholder.typicode.com/posts.
Mostra no console só os títulos (title).

4. Fetch com erro

Tenta fazer um fetch numa URL errada (ex: https://jsonplaceholder.typicode.com/abcdxyz).
Trata o erro com try/catch e mostra "Erro ao buscar dados:" + mensagem.

5. Requisições em paralelo

Cria uma função que busca usuário 1 e posts do usuário 1 ao mesmo tempo (Promise.all).
Mostra o nome do usuário e quantos posts ele tem.

6. Criando dados (POST)

Faz uma função que cria um novo post (fake) usando a API https://jsonplaceholder.typicode.com/posts.
Envia { title: "teste", body: "conteúdo do post", userId: 99 } e mostra a resposta no console.

7. Fetch com retry

Reaproveita teu fetchComRetry(url, tentativas) e testa numa URL válida e numa inválida.

Se falhar, mostra quantas tentativas foram feitas.

Se der certo, mostra o resultado.

8. Race condition

Cria duas Promises que resolvem em tempos diferentes (esperar(2000) e esperar(500)).
Usa Promise.race e mostra no console quem “ganhou”.

9. Promise.any

Simula várias Promises onde algumas rejeitam e outras resolvem.
Exemplo:

Promise.any([
  Promise.reject("falhou 1"),
  Promise.reject("falhou 2"),
  Promise.resolve("deu certo aqui!")
])


Mostra qual resultado chegou.







10. Desafio final

Cria uma função buscarUsuarioEPosts(id) que:

Busca o usuário (/users/:id)

Busca os posts dele (/posts?userId=:id)

Retorna um objeto assim:

{
  nome: "...",
  email: "...",
  posts: [ { id, title }, ... ]
}


Mostra esse objeto no console.
